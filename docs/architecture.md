# Arquitectura

- `public/` contiene el front controller.
- `routes/` define endpoints por modulo.
- `app/controllers/` contiene la logica de entrada.
- `app/validators/` valida datos de App Inventor.
- `app/services/` encapsula envio a Google Sheets y seguridad.
- `app/helpers/` centraliza JSON, XML y respuestas.
